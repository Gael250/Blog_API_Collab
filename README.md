# 📝 Blog API with Node.js + MongoDB

## 📚 Overview
This is a RESTful API for a blog platform featuring:
- User registration & login
- JWT authentication
- Role-based access (Admin / Author)
- Blog post and category management
- MongoDB schema modeling

## 🛠️ Setup

```bash
git clone <repo_url>
cd blog-api
npm install
touch .env
